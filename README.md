# Instagram-Tool

YInstAgent18 is an advanced automation tool designed to generate and test passwords for Instagram accounts efficiently. The tool utilizes input data, such as personal information, to create a comprehensive list of potential passwords. Once the password list is generated, YInstAgent18 automates the process of checking these passwords against the specified Instagram account username, providing a detailed report of incorrect and the correct password if successful.

Features:

- Intelligent Password Generation: Creates a custom list of potential passwords based on user-provided personal information (e.g., names, numbers, etc.).
- Automated Password Testing: Uses the generated password list to test login attempts for the specified Instagram account username.
- Detailed Reporting: Outputs the results in an easy-to-read file, distinguishing between incorrect and correct passwords.
- User-Friendly Interface: Designed to streamline the process with minimal input and maximum automation.
